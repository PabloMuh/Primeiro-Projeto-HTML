História Da Web

Primeiros Computadores

  • Harvard Mark(1944)
  • Colossus(1946)
  • ENIAC(1946)

Essas máquinas foram responsáveis por inaugurar uma nova era onde começariamos a abandonar as antigas práticas de fazer trabalhos dificeis e longos utilizando maneiras tradicionais e passariamos a adotar as máquinas como responsáveis pela solução desses problemas, mesmo que a ideia inicial desses computadores tenha sido para uso militar, a sua praticidade levou os seres humanos a idealizarem essas máquinas como forma de servir de uso para atividades necessárias no cotidiano.

Evolução Dos Computadores

Com a evolução dos computadores fomos capazes de trazer uma portabilidade a nivel que máquinas as quais ocupavam salas inteiras passaram a ocupar o espaço do bolso de uma calça e serem ainda capazes de realizar inúmeras tárefas sejam elas das mais complexas as mais simples, tanto é que atualmente a nossa sociedade é completamente dependente da tecnologia por causa da ampla necessidade de comunicação da atualidade, por isso que até hoje o ser humano busca inovar sua capacidade de produção de Hardware como podemos ver com a chegada dos computadores quânticos,metaverso e etc. Para que cada vez mais realidades que parecem impossíveis se tornem realidade.

Surgimento Da Internet

Antigamente o ser-humano se comunicavam a distância exclusivamente por meio de cartas, porém tudo começou a mudar com a criação do telégrafo,objeto capaz de usar a eletricidade para transmitir um mensagem em código morse para lugares muito distantes, esse foi o pontapé para o sugimento de algo muito maior que estaria por vir. Graças a o pensamento de um Homem chamado Joseph Carl que idealizou um meio de comunicação global e que seria acessível a todos, futuramente isso seria conhecido como internet e a parir dessa ideia o ser humano foi se evoluindo desde o surgimento do primeiro site, World Wibe Web(www), até a criação dos infinitos sites que temos os quais possibilitam as mais variadas tarefas.

Client/Cliente

Voltar Ao inicio

Definição

No universo da Web é possível distinguir duas principais bases sendo elas, o client e o server,o client é o hardware responsável por receber e consumir a informação que é produzida pelos servers para isso o mesmo se utiliza de funções presentes em um sistema operacional que reuni inúmeras atividades com o propósito de suprir o usuário.

Principais Ferramentas

  • Navegadores/Browsers
  • Cache

Navegadores/Browsers

Para o client poder utilizar as funções que são recebidas do server ele precisa do auxílio de programas feitos anteriormente com a função de abrir/executar arquivos, essa função é realizada pelos navegadores que possuem suas diversas variações com bases que seguem o padrão W3C, mas com a diferenciação no seu leitor de Javascript e na compatibildade de certas funções que podem tornar alguns mais favoráveis que outros dependendo da situação.

Cache

Quando o client recebe uma informação do server ele pode armazenar o que foi recebido para caso seja reutilizado as mesmas informações não seja necessário ir buscar novamente no server assim otimizando o uso dos navegadores em troca de uma quantidade de memória.

Server/Servidor

Voltar Ao inicio

Definição

Para o Cliente receber a informação que ele necessita para o seu funcionamento ele precisa de algo que seja capaz de lhe enviar esses arquivos, para isso nós usamos o server que é responsável por armazenar e enviar dados que são recebidos diretamente da internet, ou recebidos diretamente de outros computadores(caso seja um server local).

Tipos De Servidores

  • Proxy
  • Firewall
  • Email
  • Web
  • Domain Name Service(DNS)

Proxy

O proxy serve com um intermediário entre o client e o server que busca manter sua segurança ao não revelar seu verdadeiro endereço ip para os sites que acessa.

Firewall

O firewall serve como um segurança para que verifica o que entra ou sai na troca de dados entre o client e o server com restrições específicas para que nenhum arquivo malicíoso tenha acesso a sua máquina.

Email

O server de email serve para armazenar e direcionar emails para seus respectivos destinatários.

Web

Um server web é responsável pelo armazenamento, processamento e entrega dos arquivos dos sites para os navegadores.

Domain Name Service(DNS)

Um server Domain Name Service(DNS) é responsável por armazenar o ip dos sites presentes nesse servidor e direcirecionar o usuário para o site específico.
